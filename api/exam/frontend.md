## 单选题

### Q1
题型：single
难度：easy
标签：HTML
题目：HTML 的主要作用是什么？

选项：
A. 描述网页结构
B. 管理数据库连接
C. 编译服务器代码
D. 压缩图片资源

答案：A

解析：HTML 用标签描述网页内容和结构，样式通常由 CSS 负责，交互通常由 JavaScript 负责。

### Q2
题型：single
难度：easy
标签：CSS
题目：下面哪个 CSS 属性用于控制文本颜色？

选项：
A. font-size
B. color
C. display
D. position

答案：B

解析：color 控制文本颜色，font-size 控制字号，display 和 position 主要影响布局。

## 多选题

### Q3
题型：multiple
难度：medium
标签：CSS
题目：以下哪些会影响标准盒模型元素占用的宽度？

选项：
A. width
B. padding
C. border
D. color

答案：A,B,C

解析：标准盒模型下，元素占用宽度由 width、padding 和 border 共同决定，color 不影响尺寸。

### Q4
题型：multiple
难度：medium
标签：JavaScript
题目：以下哪些是 JavaScript 中常见的基本数据类型？

选项：
A. string
B. boolean
C. number
D. table

答案：A,B,C

解析：string、boolean、number 都是 JavaScript 的基础类型；table 不是 JavaScript 基本数据类型。

## 判断题

### Q5
题型：single
难度：easy
标签：JavaScript
题目：JavaScript 可以在浏览器中响应用户点击、输入等交互行为。

选项：
A. 正确
B. 错误

答案：A

解析：浏览器中的 JavaScript 常用于处理事件、更新页面状态和发起网络请求。
