### Q1
题型：single
难度：easy
标签：核心概念
题目：机器学习最核心的目标是什么？

选项：
A. 从数据中学习规律，并对新样本做出预测或决策
B. 手写所有规则覆盖所有情况
C. 只保存原始数据
D. 只设计页面布局

答案：A

解析：机器学习用数据训练模型，希望模型能在未见过的数据上泛化。

### Q2
题型：single
难度：easy
标签：任务类型
题目：预测房价通常属于哪类机器学习任务？

选项：
A. 回归
B. 分类
C. 聚类
D. 文本分词

答案：A

解析：房价是连续数值，预测连续值通常属于回归任务。

### Q3
题型：single
难度：normal
标签：过拟合
题目：过拟合指的是什么？

选项：
A. 模型在训练集表现很好，但在新数据上表现差
B. 模型完全没有训练
C. 数据没有任何特征
D. 模型只能输出网页

答案：A

解析：过拟合说明模型记住了训练数据细节或噪声，而没有学到可泛化规律。

### Q4
题型：multiple
难度：normal
标签：数据划分
题目：为什么要划分训练集、验证集或测试集？

选项：
A. 评估模型泛化能力
B. 调参时减少对测试集的污染
C. 检查模型是否只记住训练数据
D. 让数据彻底失效

答案：A,B,C

解析：合理数据划分能帮助我们判断模型在新数据上的真实表现。

### Q5
题型：single
难度：hard
标签：第一性原理
题目：机器学习项目失败时，最应该优先检查什么？

选项：
A. 问题定义、数据质量和评估指标是否正确
B. 变量名是否足够炫酷
C. README 是否有更多口号
D. 是否用了最复杂的模型

答案：A

解析：机器学习不是只换模型。问题定义、数据和指标错误时，复杂模型也无法得到有效结果。
