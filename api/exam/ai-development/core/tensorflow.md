### Q1
题型：single
难度：easy
标签：框架定位
题目：TensorFlow 主要用于什么？

选项：
A. 构建、训练和部署机器学习模型
B. 管理网页路由
C. 编写 CSS 动画
D. 只做数据库迁移

答案：A

解析：TensorFlow 是机器学习和深度学习框架，覆盖建模、训练、推理和部署生态。

### Q2
题型：single
难度：normal
标签：张量
题目：TensorFlow 中的 Tensor 通常表示什么？

选项：
A. 多维数组形式的数据
B. 网页链接
C. Git 分支
D. 操作系统进程

答案：A

解析：张量是机器学习框架处理数据的基础结构，可以表示标量、向量、矩阵和更高维数据。

### Q3
题型：multiple
难度：normal
标签：训练流程
题目：训练一个 TensorFlow 模型通常包括哪些步骤？

选项：
A. 准备数据
B. 定义模型
C. 选择损失函数和优化器
D. 训练并评估

答案：A,B,C,D

解析：机器学习训练流程通常由数据、模型、目标函数、优化和评估组成。

### Q4
题型：single
难度：normal
标签：生态
题目：TensorFlow 生态中，Keras 常用于什么？

选项：
A. 更高层、更易用地定义和训练神经网络
B. 替代操作系统
C. 压缩图片格式
D. 管理浏览器插件

答案：A

解析：Keras 提供高层 API，让模型搭建和训练流程更简洁。

### Q5
题型：single
难度：hard
标签：选型
题目：选择 TensorFlow 的常见理由是什么？

选项：
A. 需要成熟的训练和部署生态
B. 只需要写静态 HTML
C. 完全不做模型推理
D. 只处理 Excel 排版

答案：A

解析：TensorFlow 的优势之一是生态完整，适合从实验到生产部署的机器学习工作流。
